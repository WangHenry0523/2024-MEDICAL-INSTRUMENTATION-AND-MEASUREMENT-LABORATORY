# 儀表實驗課堂專題簡要內容:
## 專題內容:
運用課堂上所學到的Labview程式設計技術結合Myrio介面卡與自製PPG電路，做出一個心電圖介面程式，可以實時展現我們測量到的訊號。除此之外也添加了濾波器的功能，讓成品訊號可以更加完整，降低訊躁比。
